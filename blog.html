    • var

  • It can be declared without initialization.
  • it can re-declare and can be update variable.
  • var can also re-assign the variable
  • Var is declared globally. So, the scope of the variable is global, and it can be accessible everywhere in the program.
  • The scope of a var variable is global scope.

    • let

  • It can be declared without initialization.
  • it can re-declare and can be update variable.
  • let can't re-assign the variable
  • let is block scope. So, the scope of the variable is block, and it can't be accessible everywhere in the program.
  • The scope of a let variable is block scope.

    • const

  • It cannot be declared without initialization.
  • it can't re-declare and can't be update variable.
  • it can't re-assign the variable
  • const is block scope. So, the scope of the variable is block, and it can't be accessible everywhere in the program.
  • The scope of a const variable is block scope.

    • Regular Function

  • Inside of a regular JavaScript function, this value is dynamic.
  • the regular function can easily construct objects.
  • Inside the body of a regular function, arguments is a special array-like object containing the list of arguments with which the function has been invoked.
  • If the return statement is missing, or there's no expression after return statement, the regular function implicitely returns undefined
  • The regular functions are the usual way to define methods on classes.

    • Arrow Function

  • The behavior of this inside of an arrow function differs considerably from the regular function's this behavior. The arrow function doesn't define its own execution context.
  • A consequence of this resolved lexically is that an arrow function cannot be used as a constructor.
  • On the other side, no arguments special keyword is defined inside an arrow function.
  • If the arrow function contains one expression, and you omit the function's curly braces, then the expression is implicitly returned.
  • we can use the arrow function as methods inside classes.
